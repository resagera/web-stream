Описание проекта:
Реализация трансляции небольшого семейного события. Событие создается в небольшой админке создателем (логин и пароль у меня).
Будет несколько устройств для трансляции (ноутбук на улице, ноутбук в гостиной, планшет в кухне, мобильный телефон). Будет удалённый сервер для передачи данных. 

Будет несколько друзей, живущие в разных городах и странах, которым нужно просто дать ссылку на событие (и пароль для доступа, чтобы случайный гость не смог зайти), чтобы они могли видеть трансляции с наших 3+ устройств в одном окне и писать что-то в общий чат.
Гости могут себя идентифицировать, добавить имя и фото (в том числе с вебки сделать фото). Предложить им авторизоваться либо логином и паролем, либо уникальной ссылкой доступа. 
Зрителей ожидается не много, 20 или меньше.

#### Видео
* WebRTC для минимальной задержки.
* Сервер:
  * [LiveKit](https://livekit.io?utm_source=chatgpt.com)
* Сайт:
  * зритель заходит на сайт по ссылке события
  * Можно сделать сетку из всех камер, а зрители будут просто открывать камеру, чтобы просматривать (чатик остается на виду, при желании)
  * Нам, кто ведет трансляцию видно сколько зрителей смотрят, кто подключился в данный момент

#### Чат
* backend на Go;
* WebSocket;
